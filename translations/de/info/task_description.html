Nehmen wir eine zufällige positive ganze Zahl (z.B. 5914) und wenden wir die Kaprekar's routine darauf an: Unter erstellen Sie zunächst zwei neue Ganzzahlen, indem Sie die Ziffern in absteigender (9541) und aufsteigender Reihenfolge (1459) anordnen. Dann subtrahieren Sie die kleinere von der größeren Zahl (9541 - 1459 = 8082). Wiederholen Sie diesen Vorgang für die neue Zahl 8082.

Die Anwendung des Algorithmus von Kaprekar auf eine beliebige positive ganze Zahl kann zu drei Fällen führen:

  1. Eine konstante ganze Zahl wird erreicht
    Für 5914 lautet die Zahlenfolge: 5914, 8082, 8532, 6174, 6174, 6174, ...
    6174 ist also die Kaprekar-Konstante, die der Zahl 5914 entspricht.
  2. Ein Zyklus von ganzen Zahlen ist erreicht
    Beispiel: 48, 36, 27, 45, 09, 81, 63, 27, 45, 09, 81, 63, 27, 45, 09, 81, 63, ...
    (Zwei verschiedene Farben werden verwendet, um den Kaprekar-Kreis 27, 45, 09, 81, 63 hervorzuheben)
  3. Null wird erreicht
    Obwohl die Null technisch gesehen auch eine Konstante ist (z.B. 7, 0, 0, 0, ...), betrachten Mathematiker die Null nicht nicht als eine Kaprekar-Konstante.

Sie haben vielleicht bemerkt, dass führende Nullen eine wichtige Rolle spielen können: im Zyklus 27, 45, 09, 81, 63, muss die Neun als 09 behandelt werden, nicht nur als 9. Betrachten Sie für jedes Zwischenergebnis mit führenden Nullen auf die Gesamtzahl der Ziffern der eingegebenen ganzen Zahl auffüllen.

Eingabe: Eine positive ganze Zahl n (0 ≤ n ≤ 999.999)

Ausgabe: Ein Tupel bestehend aus drei Werten:

  1. Eine ganze Zahl, die entweder die Kaprekar-Konstante zu n sein sollte oder (im Falle eines Zyklus) die erste Zahl eines Kaprekar-Zyklus, die ab n eingegeben wird.
    Hinweis zur Klarstellung: Auch wenn die beiden ganzen Zahlen 48 und 92 zum selben Kreis führen (d.h. 27, 45, 09, 81, 63), treten beide Zahlen bei unterschiedlichen Zahlen in diesen Kreis ein. Daher sollte Ihre Funktion 27 (für n = 48) und 63 für n = 92.
  2. Die Anzahl der Kaprekar-Operationen, die auf n angewendet werden müssen, bis eine Konstante oder ein Zyklus erreicht ist. Wenn n bereits eine Kaprekar-Konstante (oder eine Zahl aus einem Kaprekar-Zyklus) ist, sollte dies als null Operationen gezählt werden.
  3. Ein Boolescher Wert oder None, der angibt, welcher Fall für n eingetreten ist: True für Fall A, False für Fall B, None für Fall C.

Beispiele: 

assert kaprekar_algorithm(5914) == (6174, 3, True)
assert kaprekar_algorithm(6174) == (6174, 0, True)
assert kaprekar_algorithm(48) == (27, 2, False)
assert kaprekar_algorithm(27) == (27, 0, False)
assert kaprekar_algorithm(45) == (45, 0, False)
assert kaprekar_algorithm(111) == (0, 1, None)
assert kaprekar_algorithm(9) == (0, 1, None)

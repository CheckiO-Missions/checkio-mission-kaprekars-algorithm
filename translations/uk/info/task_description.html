Візьмемо випадкове натуральне число (наприклад, 5914) і застосуємо до нього Kaprekar's routine: На етапі спочатку ви створюєте два нових цілих числа, впорядковуючи цифри за спаданням (9541) та зростанням (1459). Потім від більшого числа віднімається менше (9541 - 1459 = 8082). Повторіть цю процедуру для нового числа 8082.

Застосування алгоритму Капрекара до будь-якого натурального числа може призвести до трьох випадків:

  1. Буде досягнуто постійне ціле число
    Для числа 5914 послідовність чисел наступна: 5914, 8082, 8532, 6174, 6174, 6174, ...
    Таким чином, 6174 є сталою Капрекара, що відповідає 5914.
  2. Досягнуто цикл цілих чисел
    Приклад: 48, 36, 27, 45, 09, 81, 63, 27, 45, 09, 81, 63, 27, 45, 09, 81, 63, ...
    (Для виділення кола Капрекара 27, 45, 09, 81, 63 використовується два різних кольори)
  3. Досягнуто нуля
    Хоча технічно нуль також є константою (наприклад, 7, 0, 0, 0, 0, ...), математики не вважають нуль константою Капрекара.

Ви могли помітити, що ведучі нулі можуть відігравати важливу роль: у циклі 27, 45, 09, 81, 63, дев'ятка повинна розглядатися як 09, а не просто 9. Для отримання будь-якого проміжного результату подумайте про те, щоб заповнення ведучими нулями загальної кількості цифр вхідного цілого числа.

Вхідні дані: Натуральне число n (0 ≤ n ≤ 999.999)

Вихідні дані: Кортеж, що складається з трьох значень:

  1. Ціле число, яке повинно бути або константою Капрекара для n, або (у випадку циклу) першим числом a Циклу Капрекара, який вводиться з n.
    Зауваження для пояснення: Незважаючи на те, що обидва числа 48 і 92 ведуть до одного і того ж кола (тобто 27, 45, 09, 81, 63), обидва числа потрапляють у це коло під різними номерами. Таким чином, ваша функція повинна повернути 27 (для n = 48) і 63 для n = 92.
  2. Кількість операцій Капрекара, які потрібно застосувати до n, поки не буде досягнуто константи або циклу. Якщо n вже є константою Капрекара (або числом з циклу Капрекара), це слід вважати за нуль операцій.
  3. Булеве значення або None, яке вказує, який випадок мав місце для n: True для випадку A, False для випадку B, None для випадку C.

Приклади: 

assert kaprekar_algorithm(5914) == (6174, 3, True)
assert kaprekar_algorithm(6174) == (6174, 0, True)
assert kaprekar_algorithm(48) == (27, 2, False)
assert kaprekar_algorithm(27) == (27, 0, False)
assert kaprekar_algorithm(45) == (45, 0, False)
assert kaprekar_algorithm(111) == (0, 1, None)
assert kaprekar_algorithm(9) == (0, 1, None)
